## 什么是 Vibe Coding？

Vibe Coding = 用自然语言与 AI 协作编程

如今 Vibe Coding 已经火遍全网！不仅是程序员，连设计师、产品运营、甚至完全不懂技术的人都能用 Vibe Coding 实现自己的想法。

## 核心理念

> **AI 是同事和朋友，不是工具**

| 传统编程 | Vibe Coding |
|---------|-------------|
| 写代码 | 描述需求 |
| 查文档 | 问 AI |
| 单兵作战 | 人机协作 |
| 追求完美 | 快速迭代 |

## 为什么学 Vibe Coding？

✅ **零基础也能上手** — 不需要会编程  
✅ **10 倍开发效率** — 用自然语言生成代码  
✅ **降低技术门槛** — 人人都能实现想法  
✅ **AI 协作思维** — 培养人机协作能力

## 🗺️ 学习路径

### 零基础新手
- **第 1 天**：读完基础入门，理解 Vibe Coding 并做出第一个作品
- **第 1-2 周**：学习 AI 编程工具 + 做几个简单项目
- **之后**：按需学习经验技巧和产品变现

### 有编程基础
- **第 1 天**：快速过完基础内容
- **第 1 周**：学习主流 AI 编程工具，尝试重构之前的项目
- **之后**：重点学习进阶技巧，提升对话和上下文管理能力
