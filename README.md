## 什么是 Vibe Coding？

Vibe Coding = 感觉驱动编程

- 🎯 **描述需求** → AI 生成代码
- 🔧 **自然语言** → 替代命令行
- 🧪 **AI 协助调试** → 不再独自面对 bug
- 🚀 **快速原型** → 从想法到 Demo 只要几分钟

## 核心理念

| 传统编程 | Vibe Coding |
|---------|-------------|
| 写代码 | 描述需求 |
| 查文档 | 问 AI |
| 单兵作战 | 人机协作 |
| 追求完美 | 快速迭代 |

## 快速开始

```bash
# 方式 1：用 OpenClaw
openclaw init vibe-project
cd vibe-project
openclaw task "帮我写一个天气预报小程序"

# 方式 2：用 Cursor / Windsurf
# 直接在编辑器中用自然语言描述需求
```
