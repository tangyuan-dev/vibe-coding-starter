# 06. AI 数据处理

利用 AI 能力进行数据分析和处理。

## 什么是 AI 数据处理？

使用 AI 能力处理、分析和生成数据。

## 应用场景

### 数据分析
- 销售数据分析
- 用户行为分析
- 市场趋势分析

### 数据转换
- CSV 转 JSON
- 数据清洗
- 格式转换

### 数据生成
- 测试数据生成
- 模拟数据创建
- 报告生成

## 使用 AI 处理数据

### 分析 CSV 数据

```
分析这个销售数据 CSV 文件：
1. 计算每月总收入
2. 找出销量最高的产品
3. 按地区统计销售额
4. 生成可视化图表的代码

文件路径：sales_data.csv
```

### 数据清洗

```
清洗这个用户数据：
1. 删除重复记录
2. 标准化邮箱格式
3. 填补缺失的省份字段
4. 验证手机号格式

输入：users_raw.csv
输出：users_cleaned.csv
```

### 生成测试数据

```
生成 100 条用户测试数据：
字段：
- id (自增)
- name (中文姓名)
- email (真实格式)
- age (18-60)
- city (中国城市)
- created_at (过去30天内随机)

输出 JSON 格式
```

## Vibe Coding 技巧

### 让 AI 处理数据

```
帮我用 Python 处理这个数据：
1. 读取 data.xlsx
2. 筛选条件：age > 25 AND city = '北京'
3. 计算平均工资
4. 输出到 result.csv

需要先安装的依赖？
```

### 生成分析报告

```
根据这个月的销售数据，生成一份分析报告：
- 销售趋势
- 热销产品 TOP 5
- 问题和建议

用中文输出，包含数据可视化代码
```
