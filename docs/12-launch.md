## 发布流程

### 1. 准备上线

```bash
# 1. 检查代码
npm run build

# 2. 设置环境变量
# 生产环境的 API Keys 不能暴露

# 3. 测试生产构建
npm run start
```

### 2. 部署到 Vercel

```bash
# 安装 Vercel CLI
npm i -g vercel

# 部署
vercel --prod
```

### 3. 域名配置

- 购买域名（阿里云、腾讯云、Namecheap）
- 添加 DNS 解析
- 配置 HTTPS
