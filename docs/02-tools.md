# 02. AI 编程工具对比

主流 AI 编程工具横向对比，帮你选择最适合的。

## 工具分类

### 🏠 本地部署

| 工具 | 模型支持 | 特点 | 适合场景 |
|------|---------|------|---------|
| **OpenClaw** | OpenAI/Anthropic/Ollama/MiniMax | 本地运行、隐私安全、多渠道 | 自动化、助手开发 |
| **Ollama** | Llama/Mistral/Qwen | 纯本地大模型 | 实验、学习 |
| **Jan** | Llama/Qwen | 本地 AI 平台 | 个人 AI 使用 |

### ☁️ 云端服务

| 工具 | 模型 | 特点 | 价格 |
|------|------|------|------|
| **Cursor** | GPT-4/Claude | AI IDE、专业功能 | $20/月 |
| **Windsurf** | Claude | Agent 模式 | $20/月 |
| **v0** | GPT-4 | UI 生成 | 免费/付费 |
| **GitHub Copilot** | GPT-4 | 代码补全 | $10/月 |

### 🎨 UI 生成

| 工具 | 特点 | 产出 |
|------|------|------|
| **v0** | React/Vue 组件 | 代码 |
| **Lovable** | 全栈应用 | 完整项目 |
| **Bolt.new** | AI IDE | 完整项目 |

## 详细对比

### OpenClaw vs Cursor

| 维度 | OpenClaw | Cursor |
|------|----------|--------|
| 运行方式 | 本地 + 云端模型 | 云端 |
| 隐私 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| IDE 集成 | 独立应用 | VS Code 插件 |
| 自动化 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 代码编辑 | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 适用场景 | 助手开发、自动化 | 专业开发 |

### 选型建议

| 场景 | 推荐工具 |
|------|---------|
| 学习编程 | Cursor + Copilot |

| 自动化任务 | OpenClaw |

## 组合使用

最佳实践：**组合多个工具**

```
OpenClaw (任务规划 + 自动化)
    ↓
Cursor (代码编写 + 调试)
    ↓
Copilot (日常补全)
    ↓
v0 (快速 UI)
```
