## What is Vibe Coding?

Vibe Coding = Feeling-driven Programming

- 🎯 **Describe Requirements** → AI generates code
- 🔧 **Natural Language** → Replace command lines
- 🧪 **AI-assisted Debugging** → No more fighting bugs alone
- 🚀 **Rapid Prototyping** — From idea to Demo in minutes

## Core Philosophy

| Traditional Coding | Vibe Coding |
|-------------------|-------------|
| Write code | Describe requirements |
| Read docs | Ask AI |
| Solo | Human-AI collaboration |
| Perfection | Rapid iteration |

## Quick Start

```bash
# Method 1: OpenClaw
openclaw init vibe-project
cd vibe-project
openclaw task "Build a weather app"

# Method 2: Cursor / Windsurf
# Just describe your requirements in natural language
```
